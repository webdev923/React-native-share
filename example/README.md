# Getting started

# Android

To get this project running on Android just follow these steps

1. `yarn` or `npm i`
2. `yarn start` or `npm start`
3. `react-native run-android`

# iOS

To get this project running on iOS just follow these steps

1. `yarn` or `npm i`
2. `yarn start` or `npm start`
3. `react-native run-ios`
